## Easy Installation

1. Go to Slack's `Preferences` → `Sidebar Theme`
2. Scroll to the bottom and paste these colors: `#2D2B55,#6943FF,#FAD000,#1E1E3F,#6943ff,#cec5ff,#3ad900,#6943ff`
7. Done!!! 🙌
